# Mechatronics Robot Dynamics Identification

Embedded sensor-driven robotic vehicle for longitudinal system identification using MicroPython, IMU, and encoder-based experimental validation.

---

## 🚗 Sensorized Robotic Vehicle for System Identification

**Course:** ECE 566 – Mechatronics  
**Instructor:** : Christopher Pannier 

---

## 📌 Overview

This project focuses on developing an embedded, sensor-driven robotic vehicle capable of identifying its own longitudinal dynamics using onboard sensing, MicroPython, and system identification techniques.

The goal was to bridge theoretical modeling with real-world experimental validation, emphasizing sensor reliability, embedded programming, and system-level analysis.

## 🎯 Objectives

- Build a functional robotic platform with integrated sensors  
- Collect reliable real-time motion data  
- Perform system identification of longitudinal dynamics  
- Validate models against experimental results

## 🛠️ System Architecture

- Microcontroller running MicroPython  
- DC motors with encoder feedback  
- IMU for motion sensing  
- Power and motor driver circuitry
